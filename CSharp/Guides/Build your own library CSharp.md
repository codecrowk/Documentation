## Tutorial

#### Create a class library
```bash
dotnet new classlib -o StringLibrary
```

#### Make your code

#### Build the library
```bash
dotnet build
```

#### Make reference in your current project
```bash
dotnet sln add ShowCase/ShowCase.csproj
```

#### Use the library

#### Run your app
```
dotnet run --project ShowCase/ShowCase.csproj
```
